# Setup para utilizar Laravel, Svelte e Tailwind juntos 👍

Svelte: 5
<br>
Laravel: 11

## Install

```
pnpm install
```

```
cp .env.example .env
```

```
composer install
```

```
touch database/database.sqlite
```

```
php artisan key:generate
```

```
php artisan migrate
```



## Run
```
composer run dev
```
